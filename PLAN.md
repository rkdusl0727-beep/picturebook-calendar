# 그림책 달력 자동화 웹앱 계획

## 목표

새 웹앱으로 그림책 달력을 자동 생성한다. 사용자가 월별 날짜와 그림책 제목을 입력하면 네이버 책 검색 API에서 표지 이미지를 찾아 날짜 칸에 자동 배치하고, 완성된 달력을 이미지 파일로 저장한다.

## 가장 중요한 원칙

네이버 개발자 API 키 보안이 최우선이다.

- 브라우저 코드에는 `Client ID`, `Client Secret`을 넣지 않는다.
- GitHub 저장소에는 실제 키를 올리지 않는다.
- API 호출은 Netlify Function에서만 수행한다.
- 키는 로컬 `.env`와 Netlify 환경변수에만 저장한다.

## 구현 범위

1. 달력 화면
   - 월 이동
   - 연도/월 직접 입력
   - 예시 이미지와 비슷한 그림책 달력 레이아웃

2. 자동 표지 입력
   - 날짜 칸 클릭
   - 그림책 제목 입력
   - 네이버 책 검색 API에서 여러 표지 후보 표시
   - 사용자가 선택한 표지를 해당 날짜 칸에 배치
   - 보조 기능으로 `4 프레드릭` 같은 일괄 입력도 지원

3. 이미지 저장
   - 달력 영역만 PNG로 저장
   - 작업 패널은 저장 이미지에 포함하지 않음

4. 보안
   - `/api/search-books` Netlify Function 사용
   - `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET` 환경변수 사용
   - `.env` Git 제외

## 배포 흐름

1. 로컬에서 `.env` 설정
2. `npm run dev`로 테스트
3. GitHub에 코드 업로드
4. Netlify에 GitHub 저장소 연결
5. Netlify 환경변수 등록
6. 배포 후 실제 검색/이미지 저장 테스트

## 나중에 추가할 수 있는 기능

- 월별 데이터 파일 저장/불러오기
- Google Sheet에서 날짜와 제목 가져오기
- 공휴일 자동 반영
